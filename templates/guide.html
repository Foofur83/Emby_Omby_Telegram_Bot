📖 Emby Bot Web Interface - Handleiding

Welkom bij de uitgebreide handleiding voor de Emby Bot web interface. Hier vind je alles wat je nodig hebt om de bot effectief te beheren.

🏠 Dashboard

Het dashboard geeft een overzicht van alle activiteit en statistieken.

  • Totaal Aanvragen: Alle media aanvragen sinds opstart
  • In Afwachting: Aanvragen die nog niet beschikbaar zijn
  • Voltooid: Aanvragen waarbij gebruiker genotificeerd is
  • Totaal Gebruikers: Alle geregistreerde Telegram gebruikers
  • Goedgekeurde Gebruikers: Gebruikers met Emby toegang
  • In Afwachting: Registratie aanvragen die wachten op goedkeuring

💡 Tip: Het dashboard wordt automatisch bijgewerkt zodra je de pagina vernieuwt.

⚙️ Configuratie

Hier kun je alle bot instellingen aanpassen via een gebruiksvriendelijk formulier.

Telegram Instellingen

  • Bot Token: Telegram Bot API token (van @BotFather)
  • Admin Telegram ID: Jouw Telegram User ID (krijg je via @userinfobot)

Ombi Instellingen

  • Ombi URL: Volledige URL naar je Ombi server (bijv. https://ombi.domain.nl)
  • API Key: Ombi API sleutel (Settings → Ombi → API Key in Ombi)

Emby Instellingen

  • Emby URL: Volledige URL naar je Emby server (bijv. https://emby.domain.nl)
  • API Key: Emby API sleutel (Dashboard → Advanced → API Keys)

Geavanceerde Instellingen

  • Poll Interval: Seconden tussen automatische checks (standaard: 60)
⚠️ BELANGRIJK:

Na het wijzigen van configuratie moet je de bot herstarten voordat wijzigingen actief worden!

Stop de bot (Ctrl+C) en start opnieuw met python main.py

👥 Gebruikersbeheer

Beheer alle Telegram gebruikers die toegang willen tot de bot.

Nieuwe Gebruiker Goedkeuren

  1. Een gebruiker stuurt /register in de Telegram bot
  2. Jij ontvangt een notificatie als admin
  3. Ga naar Gebruikers in de web interface
  4. Vind de gebruiker in de lijst (status: ❌ Niet goedgekeurd)
  5. Vul het Emby gebruikersnaam in
  6. Klik op ✅ Goedkeuren
  7. De gebruiker ontvangt automatisch een Telegram notificatie

Gebruiker Verwijderen

Klik op 🗑️ Verwijderen naast een gebruiker om deze te verwijderen. Dit verwijdert ook alle notificatie voorkeuren.

Gebruikersinformatie

Per gebruiker zie je:

  • Telegram ID: Unieke identifier
  • Naam: Telegram voor- en achternaam
  • Username: @username (indien ingesteld)
  • Emby Account: Gekoppelde Emby gebruikersnaam
  • Status: Goedgekeurd of in afwachting
  • Registratie datum: Wanneer account aangevraagd is
💡 TIP:

Zorg dat de Emby gebruikersnaam exact overeenkomt met de gebruiker in Emby, anders kan de bot geen content afspelen!

📋 Aanvragen Beheren

Bekijk alle media aanvragen van gebruikers en beheer de lijst.

Aanvraag Informatie

Voor elke aanvraag zie je:

  • Titel: Naam van de film of serie
  • Type: Movie of Series
  • Gebruiker: Telegram ID van aanvrager
  • Status: ✅ Beschikbaar (gebruiker genotificeerd) of ⏳ In afwachting
  • Ombi ID & Emby ID: Interne IDs voor tracking
  • Aanvraag datum: Wanneer aangevraagd

Aanvragen Opschonen

Er zijn drie opties om de lijst op te schonen:

  • 🗑️ Verwijderen (per aanvraag): Verwijder één specifieke aanvraag
  • 🧹 Wis Voltooide: Verwijder alle aanvragen waar gebruikers al voor genotificeerd zijn
  • ⚠️ Wis Alles: Verwijder ALLE aanvragen (vraagt bevestiging)
⚠️ LET OP:

Het verwijderen van aanvragen is permanent! Er is geen undo functie. Maak een backup van data/requests.json voor de zekerheid.

🔒 Beveiliging & Best Practices

1. Wachtwoord Wijzigen

⚠️ BELANGRIJK: Wijzig het standaard wachtwoord direct!

  1. Open web_ui.py in een text editor
  2. Zoek naar: ADMIN_PASSWORD = "admin123"
  3. Wijzig naar een sterk wachtwoord
  4. Herstart de web interface

2. Toegang Beperken

Aanbevolen beveiligingsmaatregelen:

  • Gebruik een reverse proxy (nginx/Caddy) met HTTPS
  • Beperk toegang tot je lokale netwerk
  • Gebruik firewall regels om externe toegang te blokkeren
  • Overweeg VPN voor externe toegang

3. Backup

Maak regelmatig backups van belangrijke bestanden:

  • config.yaml - Alle configuratie
  • data/users.json - Gebruikers en voorkeuren
  • data/requests.json - Aanvragen historie
  • data/episode_notifications.json - Aflevering tracking

🐛 Problemen Oplossen

Web Interface Laadt Niet

  • Check of web_ui.py draait
  • Probeer: http://localhost:5000
  • Check firewall instellingen
  • Bekijk console voor error messages

Kan Niet Inloggen

  • Standaard wachtwoord: admin123
  • Check ADMIN_PASSWORD in web_ui.py
  • Clear browser cookies en probeer opnieuw

Configuratie Niet Opgeslagen

  • Check bestandspermissies op config.yaml
  • Zorg dat het bestand niet read-only is
  • Herstart de bot na wijzigingen!

Statistieken Kloppen Niet

  • Ververs de pagina (F5)
  • Check of JSON bestanden niet corrupt zijn
  • Herstart de web interface

🚀 Tips & Tricks

  • Dashboard in de gaten houden: Check regelmatig de "In Afwachting" statistiek om te zien of er nieuwe gebruikers of aanvragen zijn
  • Emby namen consistent: Zorg dat Telegram gebruikers hun Emby account naam weten voor snellere goedkeuring
  • Regelmatig opschonen: Gebruik "Wis Voltooide" om de aanvragenlijst overzichtelijk te houden
  • Mobile access: De interface werkt uitstekend op smartphones en tablets
  • Bookmarks: Voeg de interface toe aan je browser favorieten voor snelle toegang
